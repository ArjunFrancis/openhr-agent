### 3. **Co-Founder Matching Engine** (348 lines) - üî• **UNIQUE**
**What it does:**
- Finds co-founders with **complementary** skills (not similar!)
- Scans Y Combinator, AngelList, Indie Hackers, Twitter
- Calculates compatibility score based on:
  - Skill complementarity (35%) - Technical + Business = 100%
  - Vision alignment (25%) - Same industry/mission
  - Commitment level (20%) - Full-time vs side project
  - Personality fit (15%) - Working style match
  - Network value (5%) - Connections, reputation

**Magic Algorithm:**
```javascript
// High complementarity examples:
Technical + Business = 1.0 (perfect!)
Designer + Developer = 0.95
Sales + Product = 0.90

// Too similar:

```

**Generates introduction messages:**

**Why it matters:** No other platform does skill complementarity matching. Everyone else just shows similar people.
